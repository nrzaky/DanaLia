# UI REDESIGN - GOOGLE STYLE

Redesign the entire application using Google's design philosophy and Material Design 3 principles.

Do NOT use:

* Heavy gradients
* Glassmorphism
* Excessive shadows
* Overly decorative elements
* Generic admin dashboard appearance

The application should feel like a Google product.

Examples of inspiration:

* Google Photos
* Google Finance
* Google Tasks
* Google Material Design 3

---

# Visual Style

Design Principles:

* Clean
* Minimal
* Spacious
* Content-focused
* Friendly
* Fast
* Mobile-first

---

# Dashboard

The dashboard should resemble Google Finance.

Sections:

1. Total Savings

Large number card.

2. Savings Targets

Clean cards with progress indicators.

3. Recent Transactions

Simple list.

4. Gallery Preview

Inspired by Google Photos.

---

# Cards

Use:

* rounded-2xl
* subtle border
* very soft shadow

Cards should feel lightweight.

---

# Gallery

Inspired by Google Photos.

Requirements:

* Responsive masonry layout
* Large thumbnails
* Image preview modal
* Fast image loading

Avoid decorative frames.

---

# Forms

Inspired by Google Forms.

Requirements:

* Clean inputs
* Consistent spacing
* Clear validation
* Material Design style

---

# Tables

Inspired by Google Sheets.

Requirements:

* Clean rows
* Subtle borders
* Search
* Filters
* Pagination

---

# Animations

Very subtle.

Use:

* Fade transitions
* Smooth hover states

Avoid excessive animation.

---

# Empty States

Create meaningful empty states.

Use simple illustrations or icons.

---

# Overall Goal

The application should feel like a product designed by Google.

Focus on simplicity, clarity, usability, and whitespace.

Avoid making it look like a generic CRUD application.
